About NeoPapyrus

NeoPapyrus is an interactive digital library designed to make reading feel exploratory. Instead of a flat list of books, this page presents knowledge as a branching tree, where each glowing node becomes an entry point into a subject.

The three major branches can represent foundations, scientific frontiers, and computational tools. You can rename these clusters, move nodes around, and connect them to the actual books already served by your /book.html?book=... route.

How to use this

Replace each book slug in the links with your real NeoPapyrus book names. If you want the tree to be generated dynamically from your /api/books endpoint, the same visual layout can be hooked into JavaScript next.
