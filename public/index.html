About NeoPapyrus

NeoPapyrus is an interactive digital library designed to make reading feel exploratory. Instead of a flat list of books, this page presents knowledge as a branching tree, where each glowing connection leads to a book.

This version coordinates branches and books from the same data model, so the geometry stays aligned. It also allows you to mark selected books as recommended using a fire badge.

How to use this

Change the titles and slug values below to match your real NeoPapyrus books. To recommend a book, set recommended: true.
